# Financial Data Extractor (LLM)

An LLM-powered application that extracts structured financial information from unstructured text and news articles.

The system focuses on converting natural-language financial content into clean, machine-readable JSON output using prompt-driven extraction.

---

## What this app does

- Accepts unstructured financial text or article content
- Uses an LLM to identify key financial metrics
- Extracts values such as revenue and EPS
- Returns structured JSON output for downstream use

---

## Tech Stack

- **LangChain** – LLM orchestration
- **Groq API** – LLM inference
- **Python**
- **Jupyter Notebooks** – experimentation and prompt iteration
